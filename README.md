# LLM Response Stability and Contradiction Patterns

## Introduction
Large Language Models (LLMs) may generate varying responses even if their prompts differ slightly. This project aims to explore how different prompts affect response consistency and analyze the contradictions in generated text.

---

## Objective
This study seeks to assess:
- response stability when prompts vary
- contradiction frequency in the responses
- effect of adversarial and structured prompts on output consistency

---

## Experiment Setup
- about 45 prompts (factual + reasoning)
- 4 types of prompts per each query
- about 180 total responses evaluated

---

## Approach

### Prompt Construction
- each query rephrased as:
    - basic
    - paraphrased
    - adversarial
    - structured

### Evaluation Metrics
- semantic similarity for measuring consistency
- pairwise comparison for detecting contradictions
- manual analysis to identify conflicting statements

---

## Main Results
- stable responses noted for simple factual prompts
- higher contradiction frequency for reasoning prompts
- adversarial prompts led to inconsistent and contradictory outputs
- structured prompts helped achieve better consistency

---

## Visualization
Features:
- similarity distribution
- comparisons by prompt type
- contradiction frequency trends

---

## Conclusion
Responses generated by LLMs depend on the phrasing of prompts. Adversarial prompts may cause contradictions. Structured prompts could help improve consistency.
